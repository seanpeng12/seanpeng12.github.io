About me

大學畢業於資訊管理學系，期間專注於前後端軟體開發、資料庫設計，並接觸到 AWS 雲端技術及 Linux 系統操作。畢業專題「旅遊分析與規劃網站」提供關鍵字篩選功能，協助使用者縮短查找評論的時間，我負責整個網站的前後端開發及串接工作。這次專題經驗提升了我的技術能力，且深刻體會到團隊溝通的重要性。

大四實習於玉山銀行，擔任程式設計師。主要利用 JAVA Spring 框架開發後端 API，並與前端 JavaScript 完成整合。這段經驗讓我深入了解金融科技，特別是電子支付的技術應用，並強化了我對軟體開發中穩定性與安全性的認識。

進入資工所後，我進一步鑽研計算機結構和作業系統設計，專注於強化軟硬體整合知識。同時修習數位積體電路設計 (DIC) 與高等數位積體電路設計 (ADIC)，深入學習 IC 設計流程，從 Verilog 描述到合成、自動佈局繞線、驗證直至佈局後模擬，及使用fast-SPICE軟體進行全數位鎖相迴路 (ADPLL) 的混合訊號模擬等，培養了在數位 IC 領域的實作能力。

我的研究主題專注於隱私保護去識別化臉部辨識系統，目標是在保留辨識準確度的前提下，透過加噪聲或轉換空間技術來進行去特徵化，達到抗重建攻擊的效果。這項研究不僅提升了我的獨立解決問題能力，也讓我對資料隱私保護產生濃厚興趣。

無論在學術研究還是實習經驗中，我都深刻理解到團隊溝通與持續學習的重要性。我始終保持積極探索新技術的態度，並期待在未來工作中應用所學、持續成長，貢獻專業力量。
